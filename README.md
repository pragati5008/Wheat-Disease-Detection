
# 🌾 Wheat Disease Detection using GAN-Augmented Hybrid CNN–ViT–BiLSTM

## 📌 Overview
This project presents a deep learning-based approach for wheat disease detection and classification. It combines Convolutional Neural Networks (CNN), Vision Transformer (ViT), and BiLSTM with attention mechanisms to improve feature extraction and classification performance.

To handle dataset imbalance, Generative Adversarial Networks (GANs) are used for synthetic data generation.

---

## 🚀 Features
- Hybrid deep learning architecture (CNN + ViT + BiLSTM)
- Attention mechanism for better feature learning
- GAN-based data augmentation for class imbalance
- Improved classification performance over baseline models

---

## 🧠 Model Architecture
- CNN for spatial feature extraction  
- Vision Transformer (ViT) for global feature understanding  
- BiLSTM for sequential feature learning  
- Attention mechanism for enhanced performance
